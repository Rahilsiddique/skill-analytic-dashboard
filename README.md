# Skill Analytics Dashboard

## Tech Stack

React.js
Tailwind CSS

## Features

1. Update skills
2. Line graph to visualize your precentile
3. Pie chart to visualize marks (out of 15 here)
4. model on click of update button with inputs to collect data
